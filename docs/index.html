■  Microsoft Work IQ — Signal Schema

Speculative / Projected
This schema is speculative — it is modeled after real Microsoft Graph API patterns (/beta/reports/userInsights/, Viva Insights, Copilot telemetry), but Microsoft has not published an official "Work IQ Activity Signal" schema. Field names, enum values, and structure represent our best projection of what such an API would expose based on existing M365 telemetry patterns and Copilot integration points.
Projected fields for Work IQ / Copilot activity signals — 25 properties across identity, action, resource, agent, and metadata dimensions

■  Celonis Task Mining Sample Events — LOAN-4872

Real Schema
This schema matches the real Celonis Task Mining UiEvent format from the Chrome/Edge extension and desktop client. Fields like eventType, domPath, clipboardText, context, and extensionName are based on the actual production codebase. Only the customer data is fictional.

■  Unified Activity Schema — Normalization Target

POC Design
This is the proposed source-agnostic schema that both Work IQ and Task Mining events normalize into before AI clustering. Designed to be extensible to any agent infrastructure (LangChain, CrewAI, AutoGen). The actor_type field is the key extension over the current Task Mining schema.
Canonical schema for the clustering engine — any source adapter maps into this format
